# 选项 Options

## 示例

```yaml
Options:
  # 需要权限才能可以听到消息
  Join-Permission: 'trchat.admin'
  # 需要权限才能发送消息
  Speak-Condition: 'perm *trchat.admin'
  # 有权限时自动加入
  Auto-Join: true
  # 是否跨服
  Proxy: false
  # 是否强制跨服
  Force-Proxy: false
  # 哪些子服可以接收,写端口名,~代表所有
  Ports: 25565;...;.../~
  # 禁用Function.yml的模块
  Disabled-Functions: Mention
  # 有脏话就会禁止发送
  Filter-Before-Sending: false
  # 私聊功能
  Private: false
  # 范围
  Target: 'all/single_world/distance;10/self'
  # 二次转发(子服>跨服端>子服)
  Double-Transfer: true
```
