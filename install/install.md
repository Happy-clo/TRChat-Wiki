### 安装

* 将 TrChat.jar 文件放入服务端内 plugins 目录内
* 重启服务器即可完成安装
* <mark style="color:red;">**如若出现任何无法加载不要想着是什么TabooLib缺少等等**</mark>
* <mark style="color:red;">**都是你的操作问题,或者维护者没有处理到**</mark>

{% hint style="danger" %}
禁止使用任何的热加载插件加载 TrChat

例如：YUM、PlugMan、PlugManX和其他
{% endhint %}
