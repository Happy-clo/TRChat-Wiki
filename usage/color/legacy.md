# 普通颜色

### 用法

```
(& or §) + (0-9 or a-f)
```

### 示例

```
&atext or §btext
```

### 颜色代码

|     |     |               |
| :-: | :-: | :-----------: |
|  代码 |  名称 |      英文名      |
|  &0 |  黑色 |     black     |
|  &1 | 深蓝色 |   dark\_blue  |
|  &2 | 深绿色 |  dark\_green  |
|  &3 | 湖蓝色 |   dark\_aqua  |
|  &4 | 深红色 |   dark\_red   |
|  &5 |  紫色 |  dark\_purple |
|  &6 |  金色 |      gold     |
|  &7 |  灰色 |      gray     |
|  &8 | 深灰色 |   dark\_gray  |
|  &9 |  蓝色 |      blue     |
| \&a |  绿色 |     green     |
| \&b | 天蓝色 |      aqua     |
| \&c |  红色 |      red      |
| \&d | 粉红色 | light\_purple |
| \&e |  黄色 |     yellow    |
| \&f |  白色 |     white     |

### 格式效果

|  代码 |   格式效果  |
| :-: | :-----: |
|  §k |   随机字符  |
|  §l |  **粗体** |
|  §m | ~~删除线~~ |
|  §n |   下划线   |
|  §o |   _斜体_  |
|  §r |  重置文字样式 |
