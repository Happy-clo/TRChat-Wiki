# 渐变颜色

{% hint style="info" %}
要求：游戏版本 1.16+
{% endhint %}

### 用法

```
<g(or 'gradient')#<speed(ignorable)>:<hex range>>This is a Gradient, It's are beautifully! Do U like it?<loop(or 'l')>
```

### 示例

```
<g#2:#ffffff:#bbbbbb>text<l>
<g#2:#ffffff:#bbbbbb:#cccccc:#777777>text<l>
<g:#ffffff:#bbbbbb>text
```
